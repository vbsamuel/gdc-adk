# Negative Path Integration Prompt

```text
Review Forge-X negative paths across Stage 1 through Stage 6.

Cover at minimum:
1. malformed request or unsupported modality
2. configuration failure
3. provider transport failure
4. validation failure
5. traceability mismatch
6. replay failure
7. continuity corruption or missing snapshot
8. blocked multi-agent delegation

For each negative path, verify:
- explicit failure surface
- typed result or exception behavior
- artifact/issue/finding side effects if applicable
- observability output
- no hidden partial failure

Return:
- negative path matrix
- unhandled failures
- weak failure signaling
- exact remediation plan
```
