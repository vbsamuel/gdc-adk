# Forge-X Repo-Wide Test and Quality Pack

## Purpose

This pack defines repo-wide quality obligations beyond stage-local testing. It ensures that file coverage, subsystem integration, end-to-end scenarios, static checks, deterministic replay, and performance regression are all governed under one unified quality law.
